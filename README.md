# Osiris - Getting Started

Welcome to the Osiris Getting Started tutorial! This repository will help you quickly learn how to build AI-native data pipelines with Osiris.

## What is Osiris?

Osiris is a deterministic compiler for AI-native data pipelines by Keboola. It transforms plain English descriptions into reproducible, production-ready pipeline manifests that you can run locally or in the cloud.
